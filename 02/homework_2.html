HTML

HTML(Hypertext Markup Language)은 웹페이지에 내용을 기술하고 정의하는데 사용한다. HTML은 프로그래밍 언어가 아니고 컨텐츠의 구조를 정의하는 마크업 언어이다. HTML은 컨텐츠의 서로 다르 부분들을 씌우거나 감싸서 다른 형식으로 보이게하거나 특정한 방식으로 동작하도록 하는 일련의 요소들로 구성된다. 각 요소는 시작 태그와 종료 태그, 그리고 컨텐츠로 이루어진다.

CSS

Cascading Style Sheets(CSS)는 HTML로 작성된 문서의 표시 방법을 기술하기 위한 스타일 시트 언어로, 웹페이지 콘텐츠의 모양이나 표현을 기술하는 데 사용한다. HTML 문서의 요소들에 선택적으로 스타일을 적용할 수 있다.

JavaScript

JavaScript(JS)는 브라우저에서 실행되는 프로그램 언어이다. 사용자의 행동에 화면이 반응하는 것과 같은 동적인 기능을 웹페이지나 애플리케이션에 넣기 위해 사용한다. 자바스크립트는 Node.js를 통해 서버상에서도 구동할 수 있다.
